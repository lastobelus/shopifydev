# Shopifydev

Abstract out and port to ruby the functionality of the shopify textmate bundle for use in other editors.

## Installation

Add this line to your application's Gemfile:

    gem 'shopifydev'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shopifydev
